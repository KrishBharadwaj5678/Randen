# 🎲 Randen

Um **gerador de números aleatórios online** que permite gerar números inteiros aleatórios dentro de um intervalo personalizado.

![RandenWebDemo](https://github.com/KrishBharadwaj5678/Randen/raw/main/demos/RandenWebDemo.png)
![RandenMobileDemo](https://github.com/KrishBharadwaj5678/Randen/raw/main/demos/RandenMobileDemo.png)

## ✨ Funcionalidades

| Funcionalidade                   | Descrição                                                        |
| -------------------------------- | ---------------------------------------------------------------- |
| 🎲 Geração de números aleatórios | Gera números inteiros aleatórios dentro de um intervalo definido |
| 🔢 Intervalo personalizado       | Define valores mínimo e máximo personalizados                    |
| ⚡ Resultado instantâneo          | Gera números com apenas um clique                                |
| 📱 Design responsivo             | Otimizado para desktop, tablet e dispositivos móveis             |
| 🎨 Interface moderna             | UI limpa e intuitiva                                             |
| 🌙 Modo claro e escuro           | Suporte para temas claro e escuro                                |
| 🚀 Alto desempenho               | Aplicação leve criada com Vite para carregamento rápido          |
| 🔍 SEO otimizado                 | Inclui metadados, sitemap, robots.txt e dados estruturados       |

## 🛠️ Tecnologias utilizadas

| Tecnologia          | Finalidade                                               |
| ------------------- | -------------------------------------------------------- |
| ⚛️ **React**        | Biblioteca frontend para construção da interface         |

## 🚀 Primeiros passos

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/KrishBharadwaj5678/Randen.git
```

### 2️⃣ Entrar no diretório do projeto

```bash
cd Randen
```

### 3️⃣ Instalar dependências

```bash
npm install
```

### 4️⃣ Iniciar o servidor de desenvolvimento

```bash
npm run dev
```
