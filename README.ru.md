# 🎲 Randen

**Онлайн-генератор случайных чисел**, который позволяет генерировать случайные целые числа в заданном диапазоне.

![RandenDemo](https://github.com/KrishBharadwaj5678/Randen/raw/main/demos/RandenDemo.png)

## ✨ Возможности

| Функция                       | Описание                                                        |
| ----------------------------- | --------------------------------------------------------------- |
| 🎲 Генерация случайных чисел  | Генерация случайных целых чисел в заданном диапазоне            |
| 🔢 Пользовательский диапазон  | Настройка минимального и максимального значений                 |
| ⚡ Мгновенный результат        | Генерация чисел одним кликом                                    |
| 📱 Адаптивный дизайн          | Оптимизировано для компьютеров, планшетов и мобильных устройств |
| 🎨 Современный интерфейс      | Чистый и интуитивно понятный UI                                 |
| 🌙 Тёмная и светлая тема      | Поддержка обеих тем для удобства использования                  |
| 🚀 Высокая производительность | Лёгкое приложение, созданное с Vite для быстрой загрузки        |
| 🔍 SEO-оптимизация            | Метаданные, sitemap, robots.txt и структурированные данные      |

## 🛠️ Технологический стек

| Технология          | Назначение                                                  |
| ------------------- | ----------------------------------------------------------- |

## 🚀 Начало работы

### 1️⃣ Клонирование репозитория

```bash
git clone https://github.com/KrishBharadwaj5678/Randen.git
```

### 2️⃣ Переход в папку проекта

```bash
cd Randen
```

### 3️⃣ Установка зависимостей

```bash
npm install
```

### 4️⃣ Запуск сервера разработки

```bash
npm run dev
```
