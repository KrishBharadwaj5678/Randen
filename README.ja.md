# 🎲 Randen

カスタム範囲内でランダムな整数を生成できる**オンライン乱数生成ツール**です。

![RandenWebDemo](https://github.com/KrishBharadwaj5678/Randen/raw/main/demos/RandenWebDemo.png)
![RandenMobileDemo](https://github.com/KrishBharadwaj5678/Randen/raw/main/demos/RandenMobileDemo.png)

## ✨ 特徴

| 機能            | 説明                               |
| ------------- | -------------------------------- |
| 🎲 乱数生成       | 指定した範囲内でランダムな整数を生成               |
| 🔢 カスタム範囲     | 最小値と最大値を自由に設定可能                  |
| ⚡ 即時結果        | ワンクリックで数値を生成                     |
| 📱 レスポンシブデザイン | デスクトップ、タブレット、モバイルに最適化            |
| 🎨 モダンUI      | シンプルで直感的なユーザーインターフェース            |
| 🌙 ダーク＆ライトモード | ダークテーマとライトテーマの両方に対応              |
| 🚀 高速パフォーマンス  | Viteを使用した軽量で高速なアプリ               |
| 🔍 SEO最適化     | メタデータ、サイトマップ、robots.txt、構造化データ対応 |

## 🚀 はじめ方

### 1️⃣ リポジトリをクローン

```bash
git clone https://github.com/KrishBharadwaj5678/Randen.git
```

### 2️⃣ プロジェクトディレクトリへ移動

```bash
cd Randen
```

### 3️⃣ 依存関係をインストール

```bash
npm install
```

### 4️⃣ 開発サーバーを起動

```bash
npm run dev
```
